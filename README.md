# 👵 Cheonan-population_analysis
천안시 고령 인구 특성 분석 및 노인 복지 인프라 개선을 위한 데이터 기반 정책 제안

## 📌 1. 프로젝트 개요

천안시는 2020년 이후 지속적인 고령 인구 증가와 함께

*동남구와 서북구 간 인구 구조 차이
*의료·복지 인프라 접근성 불균형
이 점차 심화되고 있습니다.

본 프로젝트는 공공데이터를 기반으로 천안시의 고령 인구 구조를 정량적으로 분석하고, 이를 통해

*지역별 의료·복지 인프라의 부족 지점을 진단
*데이터 기반 정책 IDEA를 제시
하는 것을 목표로 수행되었습니다.



## 📁 2. 데이터 및 분석 방법
### 🔹 사용 데이터

* 천안시 행정동별 인구 데이터(연령·성별)
* 천안시 사회복지시설 현황
* 의료기관(병원·의원·약국) 위치 데이터
* 분석 기간: 2018~2025년 추세 기반

### 🔹 데이터 처리 방식 (R / tidyverse)

* 결측치·이상치 점검 및 정제
* 문자열 → 숫자형 변환
* 인구 피라미드용 tidy data 구조화
* 남성 인구는 음수 처리하여 구조적 시각화 가능
* 시설 수, 인구 대비 시설 비율 등 지표(KPI) 생성

### 🔹 주요 파생 KPI

* 의료 접근성 지표 = 병원 수 / 노년층 1만명
* 복지시설 대비 고령 비율
* 여성 고령자 비중 증가율(2018→2025)

## 💡 4. 데이터 기반 정책 제안 (IDEA)
### 🟦 IDEA 1. 방문형 복지·의료 통합 서비스 모델 제안

지역별 고령 인구 특성과 의료·복지 인프라 데이터를 종합 분석한 결과,
기존 시설 중심의 서비스만으로는 지역 간 서비스 접근성 격차를 해소하기 어렵다는 점을 확인했습니다.
이에 따라 **이동형·방문형 서비스 중심의 통합 모델** 을 제안했습니다.

#### 🔍 데이터 기반 판단


| 지역 | 데이터 분석 결과 | 제안 모델 |
|------|---------------------------|--------------------------------------------|
| **동남구** | 고령 인구 비율 ↑, 병원 밀도 ↓ → 의료 접근성 낮음 | **의료 중심 모바일 케어팀** (기초 진료·만성질환 관리·약물 관리) |
| **서북구** | 병원·의료시설 충분, 복지·돌봄 시설 ↓ | **복지·돌봄 중심 케어팀** (생활지원·치매 예방·정서 돌봄) |


➡ 데이터 기반으로 도출된 모델로, 예산·인력 한정 상황에서 지역별 특성에 맞게 자원을 최적 배분할 수 있는 방안임을 제시했습니다.

### 🟦 IDEA 2. 여성 고령층 대상 ‘맞춤형 건강검진 프로그램’ 구축

분석 결과, 천안시의 여성 고령자 비중이 지속적으로 증가하며
특히 75세 이상에서는 여성 비중이 압도적으로 높다는 점이 드러났습니다.
이에 따라 고령 여성의 건강 취약성을 반영한 성별 특화형 건강 서비스 모델을 제안했습니다.

#### 🔍 핵심 근거

여성 고령층은 골다공증·관절 질환·심혈관 질환등의 위험이 상대적으로 높습니다.
또한 평균 수명 증가에 비해 건강수명 격차가 커, **정기적 검진 수요 증가**하는 추세입니다.


**📌 제안 내용**


#### 여성 노인 특화 건강검진 패키지 개발

* 의료시설과 협렵하여 검진인프라 확보 
* 거동이 불편한 여성 노인을 위해 찾아가는 서비스 운행
* 검진 비용 지원
* 결과 연계및 사후 관리

#### 기대효과

* 여성 노인의 수명연장
* 의료비 절감
* 천안시 브랜드 가치 상승
